# 📚 Mediacité - Gestion d'une Médiathèque

Application mobile multiplateforme développée avec Flutter et Firebase
pour moderniser la gestion d'une médiathèque municipale.

## 🚀 Technologies
- Flutter 3.x / Dart 3.x
- Firebase (Auth, Firestore)
- Provider (gestion d'état)
- Architecture MVC

## 👥 Rôles utilisateurs
- 👁️ Visiteur — Accès limité au catalogue
- 👤 Usager — Accès complet, emprunts, événements
- 🔐 Admin — Gestion complète de la médiathèque

## ✅ Fonctionnalités
- Authentification (Login, Register, Admin)
- Catalogue multimédia (livres, films, magazines)
- Recherche et filtres avancés
- Espace Admin avec CRUD complet
- Espace personnel usager
- Événements culturels
- Mode visiteur

## 📱 Plateformes
Android / iOS / Web
